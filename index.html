Shortcut za Apple Wallet

Napravi Shortcut koji se aktivira automatski kada stignje Wallet notifikacija. Shortcut treba da: 

1. Trigger: "Automation" → "App" → izaberi "Wallet" → "Opens"

2. Akcije u Shortcut-u:

• Get Contents of URL:
  URL: 
  Method: POST
  Headers: Content-Type = application/json
  Body (JSON):
  {
    "amount": [iznos iz notifikacije],
    "merchant": [naziv iz notifikacije],
    "date": [danas],
    "type": "card"
  }

3. Parsiranje notifikacije (UniCredit IT):

Tekst notifikacije: "Pagamento 375 RSD Payspot"
→ Koristi "Match Text" sa regex:
   (\d[\d.,]+)\s*RSD
→ Iznos je prva grupa

Naziv: sve iza "RSD " do kraja

Nakon što Shortcut doda transakciju, otvori ovu stranicu i dodeli kategoriju ručno.
